# Useful Dynamics

Official website for Useful Dynamics - Expert consultation and development for autonomous drone systems.

## Tech Stack

- Pelican (Static Site Generator)
- Bootstrap 5 (Litera Theme)
- Python 3.10

## Local Development
```bash
python3.10 -m venv venv
source venv/bin/activate
pip install pelican markdown
make html && make serve
```

Visit: http://localhost:8000

## Deployment

Site is deployed to GitHub Pages at [usefuldynamics.io](https://usefuldynamics.io)
